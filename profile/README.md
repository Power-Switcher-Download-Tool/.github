## 📌 About the Tool
**PowerPlan Switcher** is a minimal yet powerful utility that allows you to quickly switch between Windows power profiles. It’s perfect for optimizing battery life or unleashing full system performance with one click.

## ❓ What is PowerPlan Switcher?
This tool simplifies the process of managing power modes on Windows 10 and 11. Whether you’re gaming, editing, or working on battery, you can toggle between custom or default power plans instantly without digging through settings.

## ✅ Why Choose PowerPlan Switcher?
- One-click switching between power plans  
- Create, rename, or delete custom profiles  
- Works with default and hidden system plans  
- Portable, lightweight, and fast  
- Ideal for laptops, desktops, and gaming rigs  
- Compatible with Windows 10 and Windows 11  

## 🔧 Key Features
- Instant switching via tray icon or hotkey  
- Save and load custom configurations  
- Auto-apply plan on startup or by AC/battery state  
- Detects and lists all power profiles  
- Does not modify system registry directly  
- Optional performance mode activation  
- Clean UI with zero background usage  

---

## 🖥️ System Requirements
- Windows 10/11 (64-bit)  
- 50 MB free disk space  
- Administrator rights (for plan changes)  

## ▶️ Usage
1. Download and run **PowerPlan Switcher**.  
2. Select your preferred power profile from the list.  
3. Click **Apply** to switch instantly.  
4. Optionally, create or save a custom power plan.  
5. Exit — settings persist automatically.
